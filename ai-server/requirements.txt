# ============================================================
# Local Environment Prerequisites (required for WeasyPrint PDF)
# ------------------------------------------------------------
# macOS (Homebrew):
#   brew install cairo pango gdk-pixbuf libffi harfbuzz fribidi ffmpeg mariadb
#   export DYLD_FALLBACK_LIBRARY_PATH="/opt/homebrew/lib:/usr/local/lib:$DYLD_FALLBACK_LIBRARY_PATH"
#
# Ubuntu/Debian:
#   sudo apt-get update
#   sudo apt-get install -y libcairo2 libpango-1.0-0 libgdk-pixbuf-2.0-0 libffi-dev shared-mime-info ffmpeg mariadb-server
#
# Install Python deps:
#   python -m venv .venv && source .venv/bin/activate
#   pip install -r requirements.txt
#
# Runtime notes:
# - STT(webm/mp3 -> wav) requires `ffmpeg` in PATH. (app/services/speech/stt_service.py)
# - DB runtime requires MariaDB/MySQL server reachable by DATABASE_URL.
# - Auth server module uses Java/Spring Boot separately (install JDK 17+ on local machine).
